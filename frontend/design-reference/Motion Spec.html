AlphaSwarm · Motion Spec v0.1

The swarm, in motion.

Motion in a live UI is language, not decoration. This document codifies the four animated moments that distinguish AlphaSwarm from a dashboard: what each one means, when it fires, how long it lasts, and what it must never do.

i.
Motion = state
Every animation carries meaning. If it doesn't signal something the user needs to know, it doesn't ship.
ii.
Short, not snappy
Target 180–320ms for transitions. Longer for narrative beats (shock waves). Never linear easing.
iii.
One thing at a time
When the swarm re-organizes, only the delta animates. Unchanged nodes stay still — visual noise costs trust.
iv.
Respect reduced-motion
All four moments collapse to a static state-change under prefers-reduced-motion. No ornamental fallback.
01
Agent pulse on thinking
Trigger · agent enters inference slot
Duration · 2200ms loop while active
M · 01 · pulse
Q-03 · inferring
Why it exists
Inference is the expensive, invisible step. Without a visual cue, the canvas looks frozen and users assume the sim is hung. The halo says "this agent is working."
Behavior
Two concentric rings expand outward from the node, ~2.2s cycle. Node itself gets a subtle inner glow at mid-cycle. Only fires on the node that currently holds the inference slot — at most 8 nodes animate at once (matches slot ceiling).

duration2200ms loop

opacity0.8 → 0
color--accent
02
Citation arrow on peer read
Trigger · agent cites a peer in rationale
Duration · 800ms · fires once per citation
M · 02 · citation
Q-03 · cites
I-04 · flips BUY
Why it exists
Influence is the most important invisible thing in the sim. A drawn-arrow makes agent-to-agent influence palpable — users can watch ideas propagate.
Behavior
A curved path draws from source to target using a dash-offset technique. The target node flashes on arrival (see 03). Arrow fades out over the final 250ms — it shouldn't linger and clutter the graph. Max 4 arrows drawing at once; further citations queue by 80ms.

duration800ms

glowdrop-shadow 4px

03
Signal flip flash
Trigger · agent changes BUY ↔ SELL ↔ HOLD
Duration · 320ms · fires once per flip
M · 03 · flip
Why it exists
Changing your mind is the most consequential event in a deliberation. It must feel discrete — a flicker, not a gradient. The brief white flash means: a conviction just moved.
Behavior
Node scales 1→1.4 and turns pure white for ~80ms, then snaps back to scale 1 in the new signal color. Deliberate jank — smooth tweens would undersell the event. Batched per-round so users never see more than ~12 flips flash simultaneously.
easingease-in-out (sharp)
duration320ms
peak-scale1.4×
flash-color#ffffff
batch-window250ms
04
Shock-wave propagation
Trigger · user injects disruptive event
Duration · 1800ms · one-shot
M · 04 · shock
Why it exists
Injecting a shock is the most powerful interaction in the product. It deserves a moment. The propagating ring tells the user "your disruption is spreading through the swarm" and reveals which agents react first vs. last.
Behavior
Three concentric rings expand outward from the injection point, 400ms apart. Nodes transition from BUY → thinking → SELL as the wave reaches them — delay is proportional to distance from the center. Camera does NOT shake. One-shot, no loop.
easingease-out
duration1800ms
waves3 · stagger 400ms
final-scale28×
camerano shake
