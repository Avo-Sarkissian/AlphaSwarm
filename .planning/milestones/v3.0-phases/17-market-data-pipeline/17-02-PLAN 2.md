<threat_model>
**Threat:** SSRF via crafted ticker symbol in Alpha Vantage URL.
**Mitigation:** Ticker symbols are validated against SEC symbol table in Phase 16 before reaching market_data.py. Only alphanumeric symbols pass validation. AV URL is constructed from a constant base URL with symbol as a query parameter (not path component).

**Threat:** yfinance returns manipulated/stale data.
**Mitigation:** yfinance is an unofficial Yahoo Finance scraper. Data is best-effort. The system treats all market data as informational context for LLM prompts, not for trading decisions. Degraded snapshots are clearly flagged with `is_degraded=True`.

**Threat:** Cache poisoning via tampered JSON files in data/market_cache/.
**Mitigation:** Cache files are local-only, written by the application via atomic temp-file-rename pattern. No external process writes to this directory. Pydantic `model_validate` rejects malformed data on cache read.

**Threat:** Alpha Vantage API key exposure in logs.
**Mitigation:** API key is passed as function parameter, never logged. structlog events log ticker symbol and status, not request URLs or headers containing the key.
</threat_model>

from __future__ import annotations

import asyncio
import json
from datetime import datetime, timezone

import httpx
import structlog

from alphaswarm.types import ExtractedTicker, MarketDataSnapshot

logger = structlog.get_logger(component="market_data")

CACHE_DIR = Path("data/market_cache")
CACHE_TTL_SECONDS = 3600  # 1 hour per D-10

AV_BASE_URL = "https://www.alphavantage.co/query"

# Per-ticker locks to prevent concurrent yfinance access (D-06, Pitfall 1)
_ticker_locks: dict[str, asyncio.Lock] = {}

**1. yfinance blocking fetch function (runs in thread pool per D-06):**

`def _fetch_yfinance_sync(symbol: str) -> dict:` -- a synchronous function called via `asyncio.to_thread()`.

Implementation details:
- `import yfinance as yf` (import inside function to avoid import-time side effects in tests)
- `ticker = yf.Ticker(symbol)`
- `hist = ticker.history(period="3mo")` per D-01 (90-day OHLCV)
- Convert DataFrame to list of dicts: iterate with `hist.itertuples()`, extract `date` (`.Index.isoformat()`), `open` (`float(row.Open)`), `high`, `low`, `close`, `volume` (`int(row.Volume)`)
- `info = ticker.info` per D-02
- Check for empty info: if `info.get("marketCap") is None and info.get("trailingPE") is None`, treat as yfinance failure, raise `ValueError(f"yfinance returned empty info for {symbol}")`
- Map info dict keys to our field names:
  - `pe_ratio` = `info.get("trailingPE")`
  - `market_cap` = `info.get("marketCap")`
  - `fifty_two_week_high` = `info.get("fiftyTwoWeekHigh")`
  - `fifty_two_week_low` = `info.get("fiftyTwoWeekLow")`
  - `eps_trailing` = `info.get("trailingEps")`
  - `revenue_ttm` = `info.get("totalRevenue")`
  - `gross_margin_pct` = `info.get("grossMargins")`
  - `debt_to_equity` = `info.get("debtToEquity")`
  - `company_name` = `info.get("shortName", symbol)`
- Earnings dates: wrap in try/except. `ed = ticker.earnings_dates`. If not None and not empty:
  - Most recent reported: `reported = ed[ed["Reported EPS"].notna()]`, if not empty, `latest = reported.iloc[0]`, `earnings_surprise_pct = latest.get("Surprise(%)")`
  - Next earnings: `import pandas as pd`, `now = pd.Timestamp.now(tz="America/New_York")`, `future = ed[ed.index > now]`, if not empty, `next_earnings_date = future.index[-1].isoformat()`
- Compute summary stats from price_history:
  - `last_close`: last entry's close value, or None if empty
  - `price_change_30d_pct`: if >= 22 trading days of data, `(last_close - close_30d_ago) / close_30d_ago * 100`
  - `price_change_90d_pct`: `(last_close - first_close) / first_close * 100`
  - `avg_volume_30d`: average volume of last 22 entries
- Return all data as a flat dict (not yet a MarketDataSnapshot -- caller constructs the model)

**2. Alpha Vantage async fallback (D-08, D-17, D-18):**

`async def _fetch_alpha_vantage(symbol: str, api_key: str) -> MarketDataSnapshot:`

- Use `httpx.AsyncClient(timeout=30.0)`
- Call `GLOBAL_QUOTE` endpoint: `params={"function": "GLOBAL_QUOTE", "symbol": symbol, "apikey": api_key}`
- Call `OVERVIEW` endpoint: `params={"function": "OVERVIEW", "symbol": symbol, "apikey": api_key}`
- Helper `_safe_float(val)`: returns None if val is None, "None", "-", or "0"; otherwise `float(val)`
- Map AV fields:
  - `pe_ratio` = `_safe_float(overview.get("PERatio"))`
  - `market_cap` = `_safe_float(overview.get("MarketCapitalization"))`
  - `fifty_two_week_high` = `_safe_float(overview.get("52WeekHigh"))`
  - `fifty_two_week_low` = `_safe_float(overview.get("52WeekLow"))`
  - `eps_trailing` = `_safe_float(overview.get("EPS"))`
  - `revenue_ttm` = `_safe_float(overview.get("RevenueTTM"))`
  - `gross_margin_pct` = None (AV does not provide gross margin %)
  - `debt_to_equity` = None (AV OVERVIEW does not include D/E ratio)
  - `company_name` = `overview.get("Name", symbol)`
- AV provides single-day price only (no history), so `price_history=[]`
- last_close from GLOBAL_QUOTE `"05. price"` field
- Return `MarketDataSnapshot(symbol=symbol, ..., is_degraded=False)`
- Check for AV rate limit: if response contains `"Note":` key with "Thank you for using Alpha Vantage", raise `ValueError("Alpha Vantage rate limit exceeded")`

**3. Cache read/write (D-09, D-10, D-11):**

`async def _read_cache(symbol: str, cache_dir: Path | None = None) -> MarketDataSnapshot | None:`
- `path = (cache_dir or CACHE_DIR) / f"{symbol}.json"`
- If not exists, return None
- Read with `aiofiles.open(path)`, parse JSON
- Extract `cached_at` ISO string, parse to datetime (UTC)
- If age > CACHE_TTL_SECONDS, return None (expired)
- `return MarketDataSnapshot.model_validate(data["snapshot"])`

`async def _write_cache(symbol: str, snapshot: MarketDataSnapshot, cache_dir: Path | None = None) -> None:`
- `dir_path = cache_dir or CACHE_DIR`
- `dir_path.mkdir(parents=True, exist_ok=True)`
- Payload: `{"cached_at": datetime.now(timezone.utc).isoformat(), "snapshot": snapshot.model_dump(mode="json")}`
- Atomic write: `tmp_path = dir_path / f"{symbol}.json.tmp"`, write with `aiofiles.open(tmp_path, "w")`, then `tmp_path.rename(dir_path / f"{symbol}.json")`

**4. Single-ticker fetch with fallback (D-15):**

`async def _fetch_single_ticker(symbol: str, company_name: str, av_key: str | None, cache_dir: Path | None = None) -> MarketDataSnapshot:`
- Get or create per-ticker lock from `_ticker_locks`
- `async with _ticker_locks[symbol]:`
  - Check cache first via `_read_cache(symbol, cache_dir)`
  - If cache hit, log `logger.info("cache_hit", ticker=symbol)` and return
  - Try yfinance: `data = await asyncio.to_thread(_fetch_yfinance_sync, symbol)`
  - Construct `MarketDataSnapshot` from the data dict
  - On yfinance exception: `logger.warning("yfinance_failed", ticker=symbol, error=str(exc))`
  - If AV key present, try `_fetch_alpha_vantage(symbol, av_key)`
  - On AV exception: `logger.warning("alpha_vantage_failed", ticker=symbol, error=str(exc))`
  - If both fail: `logger.warning("market_data_fetch_failed", ticker=symbol, reason="all sources failed")`
  - Return `MarketDataSnapshot(symbol=symbol, company_name=company_name, is_degraded=True)`
  - After successful fetch (yfinance or AV): `await _write_cache(symbol, snapshot, cache_dir)`
  - Return snapshot

- Results stored in a shared dict (safe because each ticker writes to a unique key)
- `async with asyncio.TaskGroup() as tg:` create one task per ticker
- Return `{symbol: snapshot}` dict
