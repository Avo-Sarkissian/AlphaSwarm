# Phase 19 -- UI Design Contract

> Visual and interaction contract for the Per-Stock TUI Consensus Display. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | Textual (Python TUI framework) |
| Preset | ALPHASWARM_THEME (custom Textual Theme, defined in tui.py:595) |
| Component library | textual.widgets (Static, Widget, Container, ScrollableContainer) |
| Icon library | Unicode block characters only (no icon font) |
| Font | Terminal default (monospace, determined by user's terminal emulator) |

**Source:** Existing codebase (tui.py), CLAUDE.md tech stack (textual >= 8.1.1)

---

## Spacing Scale

Terminal-character grid spacing (Textual uses character cells, not pixel units):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 0 chars | No gap between fill chars in bars |
| sm | 1 char | Inline separator between data fields |
| md | 2 chars | Padding inside panel borders (TCSS `padding: 1 1 0 1`) |
| lg | 1 line | Blank line separator between ticker sections |
| xl | N/A | Not used in this phase |

Exceptions: Panel width is exactly 44 characters (D-10). Bar width is exactly 12 characters (D-11). These are fixed constants, not derived from the spacing scale.

**Source:** D-10 (width=44), D-11 (BAR_WIDTH=12), existing BracketPanel pattern (BAR_WIDTH=20, padding: 1 1 0 1)

---

## Typography

Terminal TUI -- all text is monospace at terminal's configured size. Typography is controlled via Rich text styles (bold, color), not font sizes.

| Role | Style | Weight | Usage |
|------|-------|--------|-------|
| Panel title | `bold #4FC3F7` | bold | "Tickers" heading at top of panel |
| Ticker header | `bold #E0E0E0` | bold | Ticker symbol line: `AAPL  BUY  w=0.74  (68% majority)  R3` |
| Signal label (BUY) | `bold #66BB6A` | bold | Signal text in ticker header line |
| Signal label (SELL) | `bold #EF5350` | bold | Signal text in ticker header line |
| Signal label (HOLD) | `bold #78909C` | bold | Signal text in ticker header line |
| Bracket label | `#78909C` | normal (400) | Bracket name left of mini-bar (e.g. "Quants") |
| Data values | `#E0E0E0` | normal (400) | Numeric values: weighted score, majority pct, bar percentage |
| Muted/empty | `#78909C` | normal (400) | Empty state text, round labels, separator chars |

**Source:** Existing patterns -- RationaleSidebar._SIGNAL_COLORS, BracketPanel.render(), HeaderBar._render_header()

---

## Color

All colors use the established ALPHASWARM_THEME tokens. No new colors introduced.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#121212` (background) | Panel background inherited from `$panel` (#252525) |
| Secondary (30%) | `#252525` (panel) / `#1E1E1E` (surface) | Panel background fill, empty bar segments (#333333) |
| Accent (10%) | `#4FC3F7` (primary) | Panel title "Tickers" only |
| BUY signal | `#66BB6A` ($success) | BUY label text, BUY-dominant bracket bar fill |
| SELL signal | `#EF5350` ($error) | SELL label text, SELL-dominant bracket bar fill |
| HOLD signal | `#78909C` ($secondary) | HOLD label text, HOLD-dominant bracket bar fill |
| Bar empty | `#333333` | Unfilled portion of bracket mini-bars (EMPTY_CHAR) |
| Border | `$secondary` (#78909C) | Left border of panel (`border-left: solid $secondary`) |

Accent reserved for: Panel title text ("Tickers") only. Never applied to signal labels, data values, or bars.

**Source:** ALPHASWARM_THEME (tui.py:595-608), BracketPanel._SIGNAL_COLORS (tui.py:336-339), D-10 border spec

## Display Format Specification

### Per-Ticker Header Line

```
AAPL  BUY  w=0.74  (68% majority)  R3
```

| Segment | Style | Width | Alignment |
|---------|-------|-------|-----------|
| Ticker symbol | `bold #E0E0E0` | 6 chars left-padded | Left |
| Signal | `bold {signal_color}` | 4 chars | Left |
| `w=` prefix | `#78909C` | 2 chars | -- |
| Weighted score | `#E0E0E0` | 4 chars (0.XX) | Left |
| `(` + majority_pct + `% majority)` | `#78909C` | variable | Left |
| Round label | `#78909C` | 2-3 chars (`R1`/`R2`/`R3`) | Right |

Total line fits within 44-char panel width minus 2-char padding = 42 usable chars.

**Source:** D-04 (round labels), D-11 (display format), 19-CONTEXT.md specifics section

### Per-Bracket Mini-Bar Row

```
Quants        [████████░░░░] 80%
```

| Segment | Style | Width | Notes |
|---------|-------|-------|-------|
| Bracket name | `#78909C` | 14 chars left-aligned | Full display name (e.g. "Quants", "Doom-Posters") |
| `[` | `#E0E0E0` | 1 char | Bar delimiter |
| Filled chars | `{signal_color}` | 0-12 chars | Unicode full block U+2588 |
| Empty chars | `#333333` | 12-filled chars | Unicode light shade U+2591 |
| `]` | `#E0E0E0` | 1 char | Bar delimiter |
| Space + pct | `#E0E0E0` | 5 chars | e.g. ` 80%` |

BAR_WIDTH = 12 (not 20 like BracketPanel, to fit within 44-char panel).

**Source:** D-11, BracketPanel.render() pattern (tui.py:351-367)

### Per-Ticker Section Layout

```
AAPL  BUY  w=0.74  (68% majority)  R3
Quants        [████████░░░░] 80%
Degens        [██████░░░░░░] 50%
Sovereigns    [████░░░░░░░░] 33%
Macro         [██████████░░] 90%
Suits         [████████░░░░] 70%
Insiders      [██████░░░░░░] 55%
Agents        [████░░░░░░░░] 40%
Doom-Posters  [░░░░░░░░░░░░]  0%
Policy Wonks  [████████░░░░] 60%
Whales        [██████████░░] 85%
                                      <-- blank separator line
TSLA  SELL  w=0.62  (54% majority)  R3
...
```

Each ticker section = 1 header line + 10 bracket rows + 1 blank separator = 12 lines.
3 tickers = 36 lines. Panel scrolls if content exceeds viewport height.

**Source:** D-12 (scrolling rationale), D-11 (format)

## Copywriting Contract

| Element | Copy |
|---------|------|
| Panel title | `Tickers` |
| Empty state (no tickers) | `No tickers\nextracted` (two lines, styled `#78909C`) |
| Awaiting state (round in progress) | `Awaiting R{n}...` per ticker placeholder (styled `#78909C`) |
| Error state | Not applicable -- panel renders whatever data is in StateSnapshot; no error path for display-only widget |
| Destructive actions | None -- this is a read-only display panel |

**Source:** D-13 (empty state copy and color), D-03 (update after each round implies awaiting state)

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| Scroll | Vertical scroll via Textual native overflow when content exceeds panel height |
| Click | No click interaction on TickerConsensusPanel (read-only display) |
| Keyboard | No dedicated keybindings for this panel |
| Focus | `can_focus = False` (panel is passive display, not interactive) |
| Resize | Panel width stays fixed at 44 chars; height fills available space (`100%`) |
| Update frequency | 200ms poll timer (existing `_poll_snapshot` interval), diff-triggered only |

**Source:** D-12 (scroll), D-14 (diff polling), existing _poll_snapshot pattern (tui.py:851-897)

## State Contract

### New Data Types

**TickerConsensus** (frozen dataclass in state.py):

| Field | Type | Description |
|-------|------|-------------|
| `ticker` | `str` | Stock symbol (e.g. "AAPL") |
| `round_num` | `int` | Round that produced this consensus (1, 2, or 3) |
| `weighted_signal` | `str` | "BUY" / "SELL" / "HOLD" -- confidence-weighted winner |
| `weighted_score` | `float` | 0.0-1.0, normalized weighted sum for winning direction |
| `majority_signal` | `str` | "BUY" / "SELL" / "HOLD" -- discrete majority vote winner |
| `majority_pct` | `float` | 0.0-100.0, percentage of valid votes for majority signal |
| `bracket_breakdown` | `tuple[BracketSummary, ...]` | Per-bracket summary for this ticker only |

**Source:** D-05, D-06, D-08

### StateSnapshot Addition

| Field | Type | Default |
|-------|------|---------|
| `ticker_consensus` | `tuple[TickerConsensus, ...]` | `()` |

**Source:** D-06

### StateStore Addition

| Method | Signature |
|--------|-----------|
| `set_ticker_consensus` | `async def set_ticker_consensus(self, consensus: tuple[TickerConsensus, ...]) -> None` |

Pattern: identical to `set_bracket_summaries()` -- acquire `self._lock`, assign to `self._ticker_consensus`.

**Source:** D-07, existing set_bracket_summaries() (state.py:177-183)
