# Milestone v3.0: Stock-Specific Recommendations with Live Data

**Status:** ✅ SHIPPED 2026-04-08
**Phases:** 16-23
**Total Plans:** 16

## Overview

Ground the consensus cascade in real market data. Extract tickers from seed rumors, fetch live price/earnings/news data, enrich all 100 agent prompts with bracket-tailored market context, aggregate per-stock consensus with confidence-weighted voting, and surface results in the TUI and post-simulation report. Three gap-closing phases (21, 22, 23) were added after a mid-milestone audit to restore deleted validation logic, fix a pre-existing report tool name mismatch, and reconcile documentation.

### Phase 17: Market Data Pipeline

**Goal**: Before Round 1 begins, the system has fetched and cached live market data (price history, financials, earnings) for every extracted ticker, available as structured snapshots for downstream consumption
**Depends on**: Phase 16
**Requirements**: DATA-01, DATA-02, DATA-03, DATA-04
**Plans**: 3 plans

Plans:
- [x] 17-01-PLAN.md — Install yfinance, define MarketDataSnapshot model, config, test scaffolds
- [x] 17-02-PLAN.md — Core market_data.py module (yfinance fetch, cache, AV fallback, tests)
- [x] 17-03-PLAN.md — Integration wiring (simulation.py, graph.py, cli.py)

**Details:**
- MarketDataSnapshot Pydantic model: last_close, price_change_30d/90d_pct, avg_volume_30d, pe_ratio, eps, market_cap, headlines (reserved for Phase 18), degraded flag
- market_data.py: yfinance wrapped in asyncio.to_thread() with per-ticker _ticker_locks dict (not thread-safe)
- fetch_market_data() uses asyncio.TaskGroup for parallel ticker fetching
- Disk cache: atomic temp-file-rename, 1-hour TTL (CACHE_TTL_SECONDS=3600); hits logged at INFO
- Alpha Vantage fallback: GLOBAL_QUOTE + OVERVIEW endpoints; _safe_float handles None/"-"/"0"
- Neo4j: Ticker nodes merged by symbol (MERGE idempotent); HAS_MARKET_DATA edges to snapshot nodes
- Full price_history NOT stored in Neo4j — only summary stats stored
- CLI: _print_market_data_summary shown after seed injection with degraded-data warnings

## Milestone Summary

**Gap-Closing Phases:**

- Phase 21: Restore Ticker Validation (inserted after Phase 20 — audit gap TICK-02/TICK-03)
- Phase 22: Fix Report Tool Name Mismatch (inserted after Phase 20 — audit gap DRPT-01)
- Phase 23: Validation Tracking (inserted after Phase 21 — documentation gaps)

**Key Decisions:**

- yfinance wrapped in asyncio.to_thread() with per-ticker locks (not thread-safe)
- All market data fetching must complete BEFORE Round 1 begins (pre-simulation enrichment pattern)
- PromptBudgetAllocator with hard caps mandatory — token budget at 87-98% capacity before enrichment
- Alpha Vantage free tier 25 calls/day — disk caching mandatory, treat as optional fallback
- Full price_history NOT stored in Neo4j — only summary stats (last_close, price_change_30d/90d_pct, avg_volume_30d)
- Macro bracket placed in Earnings/Insider slice (locked D-04 decision)
- _lenient_parse_ticker_decisions drops malformed entries without PARSE_ERROR (backward compatibility)
- Ticker validator wired as optional callback kwarg — simulation proceeds without validation if CDN unreachable

**Issues Resolved:**

- Ticker_validator.py silently deleted during Phase 17 worktree merge (7ba7efa) — restored in Phase 21
- REACT_SYSTEM_PROMPT tool name mismatch causing "Unknown tool" errors in report — fixed in Phase 22
- REQUIREMENTS.md traceability stopping at Phase 15 — v3 requirements added in Phase 23
- VALIDATION.md tracking files with stale test names — reconciled in Phase 23

**Issues Deferred:**

- RAG vector retrieval deferred to v3.1 — market data pipeline proven stable first
- Token budget char estimates vs actual tokenizer counts — profiling deferred (qwen3.5 not yet tested at scale)
- num_ctx=4096 KV cache memory impact — profiling deferred

**Technical Debt Incurred:**

- httpx used directly in market_data.py but not declared as direct dependency (transitive via ollama) — low risk
- SeedEvent.tickers guard in simulation.py means market data is a no-op when ticker extraction produces empty list — by design but undocumented
