<threat_model>
## Threat Model (ASVS L1)

**Ticker symbol injection into Neo4j:** Ticker symbols stored on the Cycle node are `list[str]` values that have already passed SEC validation (allowlist). Only symbols present in the SEC dataset reach Neo4j. The Cypher parameterized query `$tickers` prevents injection -- Neo4j driver handles parameterization.

**CLI output injection:** Ticker symbols and company names from LLM output are displayed in the CLI summary via Python f-strings with fixed-width formatting. There is no terminal escape sequence risk because the values are short strings (1-5 chars for symbols, company names are plain text). The `_print_injection_summary()` function uses `print()` which does not interpret escape sequences.

**Validator callback None handling:** If `get_ticker_validator()` fails (e.g., SEC CDN unreachable, malformed JSON), the exception propagates up to `inject_seed()`. The caller (`_handle_inject` in cli.py) has a try/except that prints the error. Simulation does not proceed with unvalidated tickers. This is the correct behavior per D-07 -- we fail clearly on setup errors, not silently pass through invalid data.

**Graph write with empty tickers:** When all tickers fail validation, `tickers=[]` is passed to Neo4j. Neo4j handles empty list properties natively per Research Pitfall 7. No special handling needed.
</threat_model>

```python
    @staticmethod
    async def _create_cycle_with_entities_tx(
        tx: AsyncManagedTransaction,
        cycle_id: str,
        seed_rumor: str,
        overall_sentiment: float,
        entities: list[dict],  # type: ignore[type-arg]
        tickers: list[str],
    ) -> None:
        """Single transaction: create Cycle with tickers, then UNWIND Entity+MENTIONS."""
        await tx.run(
            """
            CREATE (c:Cycle {
                cycle_id: $cycle_id,
                seed_rumor: $seed_rumor,
                overall_sentiment: $overall_sentiment,
                tickers: $tickers,
                created_at: datetime()
            })
            """,
            cycle_id=cycle_id,
            seed_rumor=seed_rumor,
            overall_sentiment=overall_sentiment,
            tickers=tickers,
        )
        # Entity UNWIND unchanged
        if entities:
            await tx.run(
                """
                UNWIND $entities AS e
                MERGE (entity:Entity {name: e.name, type: e.type})
                WITH entity, e
                MATCH (c:Cycle {cycle_id: $cycle_id})
                CREATE (c)-[:MENTIONS {relevance: e.relevance, sentiment: e.sentiment}]->(entity)
                """,
                entities=entities,
                cycle_id=cycle_id,
            )
```
