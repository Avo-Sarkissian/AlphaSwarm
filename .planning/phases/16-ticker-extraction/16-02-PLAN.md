<threat_model>
## Threat Model (ASVS L1)

**Data tampering / malformed JSON from SEC CDN:** The SEC CDN serves public JSON over HTTPS. A MITM attack could serve malformed data, but HTTPS prevents this in practice. Mitigation: `json.loads()` + dict comprehension with `entry["ticker"]` access will raise KeyError on unexpected schema, preventing silent corruption. The `response.raise_for_status()` call rejects non-200 responses.

**Partial write on interrupted download:** If the process is killed during `dest.write_bytes(response.content)`, the file could be partially written. Mitigation: `response.content` loads the full response into memory before writing (2.5MB is negligible). The `write_bytes()` call is atomic at the OS level for small files on APFS. On next run, the partial file would fail `json.loads()` and the download would not re-trigger (file exists check). To be safe, the executor should use write-to-temp-then-rename pattern: write to `sec_tickers.json.tmp`, then `Path.rename()` to `sec_tickers.json`. This makes the file appear atomically.

**Ticker symbol injection prevention:** The validator builds a `set[str]` of uppercase symbols from SEC data. Only symbols in this set pass validation. This acts as a strict allowlist -- any LLM-fabricated symbol (e.g., "XYZFAKE", "../../etc/passwd") will not be in the SEC set and will be rejected. The `symbol.upper()` normalization prevents case-based bypasses.

**SSRF via SEC URL:** The SEC_TICKERS_URL is hardcoded as a string constant. No user input flows into the URL. SSRF is not applicable.

**Resource exhaustion:** The SEC file is ~2.5MB. The in-memory `set[str]` of ~10K short strings uses ~100KB. No memory concern on M1 Max 64GB.
</threat_model>

```python
"""SEC ticker symbol validation for AlphaSwarm.

Loads company_tickers.json from SEC CDN (one-time download),
caches as in-memory set for O(1) symbol lookup.
"""

from __future__ import annotations

import json
from pathlib import Path
from typing import Callable

import httpx
import structlog

logger = structlog.get_logger(component="ticker_validator")

SEC_TICKERS_URL = "https://www.sec.gov/files/company_tickers.json"
SEC_USER_AGENT = "AlphaSwarm admin@alphaswarm.local"
DEFAULT_DATA_DIR = Path("data")
DEFAULT_FILENAME = "sec_tickers.json"

# Module-level cache (lazy-loaded, process-lifetime)
_ticker_set: set[str] | None = None


def _load_ticker_set_from_file(path: Path) -> set[str]:
    """Parse SEC JSON into uppercase ticker symbol set."""
    with path.open() as f:
        data = json.load(f)
    return {entry["ticker"].upper() for entry in data.values()}


async def _download_sec_tickers(dest: Path) -> None:
    """Download company_tickers.json from SEC CDN.

    Uses atomic write (temp file + rename) to prevent partial file on interrupt.
    """
    dest.parent.mkdir(parents=True, exist_ok=True)
    tmp_path = dest.with_suffix(".json.tmp")
    async with httpx.AsyncClient() as client:
        response = await client.get(
            SEC_TICKERS_URL,
            headers={"User-Agent": SEC_USER_AGENT},
            timeout=30.0,
        )
        response.raise_for_status()
        tmp_path.write_bytes(response.content)
    tmp_path.rename(dest)
    logger.info("sec_tickers_downloaded", path=str(dest), size_bytes=dest.stat().st_size)


async def ensure_sec_data(data_dir: Path | None = None) -> Path:
    """Ensure SEC tickers file exists on disk. Download if missing.

    Args:
        data_dir: Override for the data directory. Defaults to project-root data/.

    Returns:
        Path to the SEC tickers JSON file.
    """
    directory = data_dir or DEFAULT_DATA_DIR
    path = directory / DEFAULT_FILENAME
    if not path.exists():
        logger.info("sec_tickers_missing_downloading", path=str(path))
        await _download_sec_tickers(path)
    return path


async def get_ticker_validator(
    data_dir: Path | None = None,
) -> Callable[[str], bool]:
    """Return a sync validation function: symbol -> bool.

    Downloads SEC data if missing. Caches ticker set in module-level variable.
    Thread-safe for read after initial load (set is immutable once assigned).

    Args:
        data_dir: Override for the data directory (useful for testing).

    Returns:
        A function that takes a ticker symbol string and returns True if valid.
    """
    global _ticker_set
    if _ticker_set is None:
        path = await ensure_sec_data(data_dir)
        _ticker_set = _load_ticker_set_from_file(path)
        logger.info("sec_tickers_loaded", count=len(_ticker_set))

    def validate(symbol: str) -> bool:
        return symbol.upper() in _ticker_set  # type: ignore[union-attr]

    return validate
```

Key implementation details:
- `SEC_USER_AGENT` header is REQUIRED by SEC (403 without it) per Research Pitfall 1
- `symbol.upper()` normalization in `validate()` handles case-insensitive LLM output per Research Pitfall 6
- Atomic write via `.tmp` + `rename()` prevents partial file corruption per threat model
- `data_dir` parameter allows test injection of `tmp_path` directories
- Module-level `_ticker_set` cache with `global` matches the project's singleton pattern
