| Property | Value |

## Sampling Rate

- **After every task commit:** Run `cd frontend && npm run check` (TS gate; ~5s)
- **After every plan wave:** Run `cd frontend && npm run check && npm run build` (full Vite production build; ~30s)
- **Before `/gsd-verify-work`:** Full suite + production bundle grep + manual UAT against live backend per parity matrix
- **Max feedback latency:** ~5 seconds per task; ~30 seconds per wave
