**Step 1 — Add _handle_web function (per D-12, D-13):**

Add the following function BEFORE the `main()` function (after `_handle_replay`, around line 817):

```python
def _handle_web(host: str, port: int) -> None:
    """Start Uvicorn web server. Synchronous — Uvicorn manages the event loop.

    Per D-13: uvicorn.run() is a blocking synchronous call that creates and owns
    the asyncio event loop internally. Do NOT wrap with asyncio.run().
    All stateful objects are created inside the FastAPI lifespan context (D-04).
    """
    import uvicorn

    from alphaswarm.web import create_app

    uvicorn.run(create_app(), host=host, port=port)
```

Key points:
- `def` not `async def` — uvicorn.run() is synchronous (per D-13, Pitfall 2)
- Lazy imports inside the function body (matches existing CLI pattern: _handle_tui, _handle_run, etc.)
- No `asyncio.run()` wrapper
- No `--reload` argument (per D-12)
