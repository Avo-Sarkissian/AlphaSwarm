    **Extend `src/alphaswarm/interview.py`** -- add `InterviewEngine` class below the dataclasses:

    ```python
    class InterviewEngine:
        """Manages multi-turn agent interview conversation.

        Per D-08: 10-turn sliding window (10 user+agent pairs = 20 messages).
        Per D-09: Summary of dropped pairs prepended as system message.
        Per D-10: Summary generated via worker model (same model as interview).
        Per D-13: Uses OllamaClient.chat() directly (no governor).
        """

        WINDOW_SIZE = 10  # max user+agent message pairs

        def __init__(
            self,
            context: InterviewContext,
            ollama_client: OllamaClient,
            model: str,
        ) -> None:
            self._context = context
            self._client = ollama_client
            self._model = model
            self._history: list[dict[str, str]] = []
            self._summary: str | None = None
            self._log = structlog.get_logger(component="interview")
    ```

    **`_build_system_prompt()`** method returns the agent's `interview_system_prompt` from context (already stripped of JSON instructions).

    **`_build_context_block()`** method returns a formatted string:
    ```
    === Your Simulation Context ===

    Narrative Summary:
    {decision_narrative}

    Round-by-Round Decisions:
    Round {n}: Signal={signal}, Confidence={confidence}, Sentiment={sentiment}
    Rationale: {rationale}
    [repeated for each RoundDecision]
    ```

    **`_build_messages()`** method assembles the full message list for OllamaClient.chat():
    1. `{"role": "system", "content": self._build_system_prompt()}`
    2. `{"role": "system", "content": self._build_context_block()}`
    3. If `self._summary` is not None: `{"role": "system", "content": f"[Earlier: {self._summary}]"}` (per D-09)
    4. All messages from `self._history`

    **`async def ask(self, user_message: str) -> str`** method:
    1. Append `{"role": "user", "content": user_message}` to `self._history`
    2. Build messages via `self._build_messages()`
    3. Call `response = await self._client.chat(model=self._model, messages=messages, think=False)`
    4. Extract `assistant_content = response.message.content or ""`
    5. Append `{"role": "assistant", "content": assistant_content}` to `self._history`
    6. Call `await self._trim_window()`
    7. Return `assistant_content`

    **`async def _trim_window(self) -> None`** method:
    1. Count pairs: `pair_count = len(self._history) // 2`
    2. If `pair_count <= self.WINDOW_SIZE`: return (nothing to trim)
    3. Extract oldest pair: `dropped_user = self._history[0]`, `dropped_assistant = self._history[1]`
    4. Remove oldest pair: `self._history = self._history[2:]`
    5. Generate summary via worker model:
       ```python
       summary_prompt = [
           {"role": "system", "content": "In one sentence, summarize what the user asked and what you said in the following exchange."},
           {"role": "user", "content": dropped_user["content"]},
           {"role": "assistant", "content": dropped_assistant["content"]},
           {"role": "user", "content": "Summarize the above exchange in one sentence."},
       ]
       summary_response = await self._client.chat(
           model=self._model, messages=summary_prompt, think=False
       )
       new_summary = summary_response.message.content or ""
       ```
    6. Merge with existing summary: if `self._summary` is not None, set `self._summary = f"{self._summary} {new_summary}"` else `self._summary = new_summary`
    7. Log: `self._log.debug("window_trimmed", pair_count=len(self._history) // 2, has_summary=True)`

    Import `OllamaClient` at the top of interview.py (runtime import, not TYPE_CHECKING -- it's used in InterviewEngine constructor type hint and at runtime).
