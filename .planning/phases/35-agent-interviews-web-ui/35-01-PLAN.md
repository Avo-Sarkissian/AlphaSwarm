<threat_model>
ASVS L1 assessment for POST /api/interview/{agent_id}:

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Prompt injection via user message | LOW | InterviewEngine already handles this -- user message is a chat turn, not a system prompt. The agent's system_prompt is reconstructed from Neo4j, not from user input. No additional mitigation needed. |
| Unbounded session accumulation | LOW | interview_sessions dict grows by one InterviewEngine per unique agent_id (max 100 agents). Each engine holds ~10 message pairs in memory. Total: ~100 * 10 * 2KB = ~2MB. Acceptable for M1 64GB. Sessions clear on new simulation start (see Task 2). |
| agent_id path traversal | LOW | agent_id is used as a dict key and passed to read_agent_interview_context() which uses parameterized Cypher queries. No filesystem access. No injection vector. |
| DoS via rapid requests | LOW | Each request awaits InterviewEngine.ask() which calls Ollama sequentially. Natural rate limit from LLM inference time (~2-10s per request). No amplification. |

No HIGH severity threats identified. All threats are LOW severity with existing mitigations.
</threat_model>

    ```python
    """Interview endpoint for post-simulation agent Q&A."""

    from __future__ import annotations

    import structlog
    from fastapi import APIRouter, HTTPException, Request, status
    from pydantic import BaseModel

    from alphaswarm.interview import InterviewEngine

    log = structlog.get_logger(component="web.interview")

    router = APIRouter()


    class InterviewRequest(BaseModel):
        message: str


    class InterviewResponse(BaseModel):
        response: str


    @router.post("/interview/{agent_id}", response_model=InterviewResponse)
    async def interview_agent(
        agent_id: str, body: InterviewRequest, request: Request,
    ) -> InterviewResponse:
        """Send a message to an agent and receive their in-character response (per D-08).

        On first call for an agent_id:
        1. Resolve cycle_id from the most recent completed cycle
        2. Reconstruct InterviewContext from Neo4j
        3. Create InterviewEngine with worker model
        4. Store in app.state.interview_sessions for reuse

        Subsequent calls reuse the existing engine (per D-06).
        """
        app_state = request.app.state.app_state
        graph_manager = app_state.graph_manager
        ollama_client = app_state.ollama_client

        # Guard: services must be available (mirrors replay.py 503 pattern)
        if graph_manager is None or ollama_client is None:
            raise HTTPException(
                status_code=status.HTTP_503_SERVICE_UNAVAILABLE,
                detail={"error": "services_unavailable", "message": "Neo4j or Ollama is not connected"},
            )

        sessions: dict[str, InterviewEngine] = request.app.state.interview_sessions

        if agent_id not in sessions:
            # Resolve cycle_id: most recent completed cycle
            cycles = await graph_manager.read_completed_cycles(limit=1)
            if not cycles:
                raise HTTPException(
                    status_code=status.HTTP_404_NOT_FOUND,
                    detail={"error": "no_completed_cycle", "message": "No completed simulation cycle found"},
                )
            cycle_id = cycles[0]["cycle_id"]

            # Reconstruct agent context from Neo4j
            context = await graph_manager.read_agent_interview_context(agent_id, cycle_id)

            # Create engine with worker model alias from settings
            engine = InterviewEngine(
                context=context,
                ollama_client=ollama_client,
                model=app_state.settings.ollama.worker_model_alias,
            )
            sessions[agent_id] = engine
            log.info("interview_session_created", agent_id=agent_id, cycle_id=cycle_id)

        engine = sessions[agent_id]
        response_text = await engine.ask(body.message)
        log.info("interview_turn", agent_id=agent_id, message_len=len(body.message))
        return InterviewResponse(response=response_text)
    ```
