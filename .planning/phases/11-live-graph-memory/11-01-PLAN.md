2. **Create src/alphaswarm/write_buffer.py** (per D-02: standalone, separate from GraphStateManager):

```python
"""Write-behind buffer for RationaleEpisode graph writes (D-01, D-02).

Agents push EpisodeRecords into an asyncio.Queue as they complete inference.
A single flush() call drains the queue and delegates batch writes to
GraphStateManager. TUI sees per-agent updates immediately via StateStore;
Neo4j receives efficient batch writes at round boundaries.

ORDERING CONSTRAINT: flush() MUST be called AFTER write_decisions() completes.
The MATCH on Decision {decision_id} in the episode write requires the Decision
node to already exist in Neo4j.
"""
```

Module contents:
- `EpisodeRecord` frozen dataclass with fields:
  - `decision_id: str` -- links to parent Decision node for HAS_EPISODE edge
  - `agent_id: str`
  - `rationale: str`
  - `peer_context_received: str` -- full formatted context string (empty string "" for Round 1 per Pitfall 3)
  - `flip_type: str` -- FlipType enum .value string
  - `round_num: int`
  - `cycle_id: str`

- `compute_flip_type(prev_signal: SignalType | None, curr_signal: SignalType) -> FlipType`:
  - If `prev_signal is None` or `prev_signal == SignalType.PARSE_ERROR`: return `FlipType.NONE`
  - If `curr_signal == SignalType.PARSE_ERROR`: return `FlipType.NONE`
  - If `prev_signal == curr_signal`: return `FlipType.NONE`
  - Otherwise: `key = f"{prev_signal.value}_to_{curr_signal.value}"` and return `FlipType(key)`

- `WriteBuffer` class:
  - `__init__(self, maxsize: int = 200)`:
    - `self._queue: asyncio.Queue[EpisodeRecord] = asyncio.Queue(maxsize=maxsize)`
    - `self._log = structlog.get_logger(component="write_buffer")`
  - `async push(self, record: EpisodeRecord) -> None`:
    - Try `self._queue.put_nowait(record)`
    - On `QueueFull`: drain one with `get_nowait()`, then `put_nowait(record)` (same drop-oldest pattern as StateStore)
  - `def drain(self) -> list[EpisodeRecord]`:
    - Loop `get_nowait()` until `QueueEmpty`, return collected list
  - `async flush(self, graph_manager, entity_names: list[str]) -> int`:
    - Call `self.drain()`
    - If empty: log debug, return 0
    - Call `await graph_manager.write_rationale_episodes(records)` passing drained records
    - Call `await graph_manager.write_narrative_edges(records, entity_names)` for REFERENCES edge creation
    - Log info with count, return `len(records)`

Use TYPE_CHECKING guard for GraphStateManager import to avoid circular imports:
