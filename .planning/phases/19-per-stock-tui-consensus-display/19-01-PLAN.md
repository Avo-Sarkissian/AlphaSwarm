   def compute_ticker_consensus(
       agent_decisions: list[tuple[str, AgentDecision]] | tuple[tuple[str, AgentDecision], ...],
       personas: list[AgentPersona],
       brackets: list[BracketConfig],
       influence_weights: dict[str, float],

   ) -> tuple[TickerConsensus, ...]:

2. Implementation logic (per D-08, D-09):
   - Build `persona_base: dict[str, float] = {p.id: p.influence_weight_base for p in personas}`
   - Collect all tickers from `decision.ticker_decisions` across all non-PARSE_ERROR agents
   - For each ticker (sorted alphabetically):
     - Init `majority_counts: dict[str, int] = {"BUY": 0, "SELL": 0, "HOLD": 0}`
     - Init `weighted_sums: dict[str, float] = {"BUY": 0.0, "SELL": 0.0, "HOLD": 0.0}`
     - Init `ticker_agent_subset: list[tuple[str, AgentDecision]] = []`
     - For each `(agent_id, decision)`:
       - Skip if `decision.signal == SignalType.PARSE_ERROR`
       - Skip if `not decision.ticker_decisions`
       - Find `td = next((t for t in decision.ticker_decisions if t.ticker == ticker), None)`
       - Skip if `td is None`
       - Append to `ticker_agent_subset`
       - `direction = td.direction.value.upper()` (converts "buy" -> "BUY")
       - `w = influence_weights.get(agent_id, persona_base.get(agent_id, 1.0))`
       - `majority_counts[direction] += 1`
       - `weighted_sums[direction] += decision.confidence * w`
     - Compute results:
       - `total_votes = sum(majority_counts.values())`
       - `total_weighted = sum(weighted_sums.values())`
       - `majority_signal = max(majority_counts, key=lambda k: majority_counts[k])` (tie-break: first max wins, alphabetical order gives BUY > HOLD > SELL)
       - `weighted_signal = max(weighted_sums, key=lambda k: weighted_sums[k])`
       - `majority_pct = majority_counts[majority_signal] / total_votes * 100.0 if total_votes > 0 else 0.0`
       - `weighted_score = weighted_sums[weighted_signal] / total_weighted if total_weighted > 0.0 else 0.0`
     - Call `bracket_breakdown = compute_bracket_summaries(ticker_agent_subset, personas, brackets)` for ticker-scoped bracket data
     - Build `TickerConsensus(ticker=ticker, round_num=round_num, weighted_signal=weighted_signal, weighted_score=weighted_score, majority_signal=majority_signal, majority_pct=majority_pct, bracket_breakdown=bracket_breakdown)`
   - Return `tuple(result)`
