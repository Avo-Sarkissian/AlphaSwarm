```python
ALPHASWARM_CHART_STYLE = Style(
    background="#121212",
    plot_background="#1E1E1E",
    foreground="#E0E0E0",
    foreground_strong="#FFFFFF",
    foreground_subtle="#78909C",
    colors=("#66BB6A", "#EF5350", "#78909C"),
    font_family="'Segoe UI', 'Helvetica Neue', Arial, sans-serif",
    label_font_size=12,
    title_font_size=14,
    value_font_size=11,
    guide_stroke_color="#333333",
    major_guide_stroke_color="#444444",
)

CHART_CONFIG: dict[str, object] = {
    "style": ALPHASWARM_CHART_STYLE,
    "disable_xml_declaration": True,
    "js": [],
    "show_legend": True,
    "print_values": True,
    "explicit_size": False,
}
```

Functions (per D-01, D-02, D-03, RESEARCH Patterns 1-3):

1. `render_consensus_bar(consensus_data: dict[str, object]) -> str`:
   - Return `""` if `consensus_data.get("total", 0) == 0`
   - Create `pygal.HorizontalBar(**CHART_CONFIG, width=600, height=300)`
   - Set `chart.title = "Final Consensus (Round 3)"`
   - `chart.add("BUY", consensus_data["buy_count"])`
   - `chart.add("SELL", consensus_data["sell_count"])`
   - `chart.add("HOLD", consensus_data["hold_count"])`
   - Return `chart.render(is_unicode=True)`

2. `render_round_timeline(timeline_data: list[dict[str, object]]) -> str`:
   - Return `""` if `not timeline_data`
   - Create `pygal.Line(**CHART_CONFIG, width=600, height=300, dots_size=4)`
   - Set `chart.title = "Signal Distribution by Round"`
   - Set `chart.x_labels = [f"Round {r['round_num']}" for r in timeline_data]`
   - `chart.add("BUY", [r["buy_count"] for r in timeline_data])`
   - `chart.add("SELL", [r["sell_count"] for r in timeline_data])`
   - `chart.add("HOLD", [r["hold_count"] for r in timeline_data])`
   - Return `chart.render(is_unicode=True)`

3. `render_bracket_breakdown(bracket_data: list[dict[str, object]]) -> str`:
   - Return `""` if `not bracket_data`
   - Create `pygal.HorizontalStackedBar(**CHART_CONFIG, width=600, height=400)`
   - Set `chart.title = "Bracket Consensus Breakdown"`
   - Set `chart.x_labels = [b["bracket"] for b in bracket_data]`
   - `chart.add("BUY", [b["buy_count"] for b in bracket_data])`
   - `chart.add("SELL", [b["sell_count"] for b in bracket_data])`
   - `chart.add("HOLD", [b["hold_count"] for b in bracket_data])`
   - Return `chart.render(is_unicode=True)`

4. `render_ticker_consensus(ticker_data: dict[str, object]) -> str`:
   - Return `""` if `not ticker_data.get("ticker")`
   - Create `pygal.HorizontalBar(**CHART_CONFIG, width=400, height=200, margin=10)`
   - Set `chart.title = f"{ticker_data['ticker']} Consensus"`
   - Parse majority_pct: `pct = float(ticker_data.get("majority_pct", 0)) * 100`
   - Map signal to series: if majority_signal is "BUY", add BUY=pct, SELL=0, HOLD=100-pct; similarly for SELL/HOLD
   - Return `chart.render(is_unicode=True)`
