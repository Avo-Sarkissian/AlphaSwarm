## Color

### Page-Level Color Contract (60/30/10 Split)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#121212` | Page `<body>` background -- deepest dark surface |
| Secondary (30%) | `#1E1E1E` | Chart panels (`.chart-panel`), section cards (`.section`), table backgrounds |
| Accent (10%) | `#4FC3F7` | Page title (h1) text, section heading (h2) text, report branding |

## pygal Chart Style Contract

All charts use a single shared `Style` instance:

```python
ALPHASWARM_CHART_STYLE = Style(
    background="#121212",
    plot_background="#1E1E1E",
    foreground="#E0E0E0",
    foreground_strong="#FFFFFF",
    foreground_subtle="#78909C",
    colors=("#66BB6A", "#EF5350", "#78909C"),  # BUY, SELL, HOLD
    font_family="'Segoe UI', 'Helvetica Neue', Arial, sans-serif",
    label_font_size=12,
    title_font_size=14,
    value_font_size=11,
    guide_stroke_color="#333333",
    major_guide_stroke_color="#444444",
)
```

All charts use shared config overrides:

| Config Key | Value | Reason |
|------------|-------|--------|
| `disable_xml_declaration` | `True` | Inline SVG in HTML -- no XML prolog |
| `js` | `[]` | Self-contained requirement -- no external JS |
| `show_legend` | `True` | BUY/SELL/HOLD series need labels |
| `print_values` | `True` | Show numeric values on bars/points |
| `explicit_size` | `False` | SVG scales to container via CSS |

Source: RESEARCH.md Pattern 1, RESEARCH.md Pitfall 1 and 2.
