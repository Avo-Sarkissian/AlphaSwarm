## What Was Built

### `src/alphaswarm/web/simulation_manager.py`

Four precise changes:

1. **Imports:** Added `from collections.abc import Awaitable` and `import httpx`; added `from fastapi import FastAPI` under `TYPE_CHECKING`.

2. **`_auto_trigger_advisory(app, cycle_id)` module-level coroutine:** Uses `httpx.ASGITransport(app=app)` to POST `/api/advisory/{cycle_id}` in-process (no network socket). Swallows 202 (log INFO), 409 (log INFO), 503 (log WARNING), unexpected statuses (log WARNING), and any network-level exception (log exception + swallow). Top-level `try/except` ensures this task never surfaces as an unhandled exception.

3. **`SimulationManager.__init__`:** Added `on_complete: Callable[[str], Awaitable[None]] | None = None` parameter (backward-compatible — existing callers in `app.py` unchanged). Stores as `self._on_complete`. Initializes `self._last_cycle_id: str | None = None`.

4. **`SimulationManager._run()`:** Captures `result = await run_simulation(...)` and sets `self._last_cycle_id = result.cycle_id` after successful return.

5. **`SimulationManager._on_task_done()`:** Replaces the old if/elif/else log block. On success (not cancelled, no exception): reads `self._last_cycle_id`, resets it to `None`, then calls `asyncio.create_task(self._on_complete(cycle_id), name=f"auto_advisory_{cycle_id}")` if both are non-None. Cancelled and exception paths are unchanged in behavior.
