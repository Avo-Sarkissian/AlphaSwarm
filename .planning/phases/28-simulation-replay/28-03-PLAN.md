    **Prerequisites:** A completed simulation cycle must exist in Neo4j. If none exists, run a quick simulation first: `uv run alphaswarm tui` and enter a seed rumor, wait for completion.

    **Test 1: CLI replay entry**
    1. Run `uv run alphaswarm replay` (no --cycle flag)
    2. Confirm TUI launches showing the replay of the most recent cycle
    3. Confirm header shows amber "REPLAY -- Cycle {8-char-id}" badge
    4. Confirm footer shows `Mode: Replay | Cycle: ... | Space/Right: Next  P: Play/Pause  Esc: Exit`

    **Test 2: Auto-advance timer**
    1. Observe rounds advancing automatically approximately every 3 seconds
    2. Confirm grid cells update to correct signals per round (colors change between rounds)
    3. Confirm bracket panel updates per round
    4. Confirm rationale sidebar updates per round
    5. After Round 3, confirm header shows "[DONE]" and notification "Replay complete -- Esc to exit"

    **Test 3: Manual mode toggle**
    1. Start a new replay (exit and re-enter, or press Esc then `r` from COMPLETE state)
    2. Immediately press `P` to enter manual mode
    3. Confirm header shows "[PAUSED]" and notification "Manual mode -- Space or Right to advance"
    4. Press `Space` -- confirm single round advance (Round 1 -> 2)
    5. Press `Right arrow` -- confirm another advance (Round 2 -> 3)
    6. Press `P` again -- confirm "[AUTO]" resumes if not at Round 3

    **Test 4: Replay exit and state restoration**
    1. During replay, press `Escape`
    2. Confirm header reverts to standard "Complete" format
    3. Confirm footer reverts to telemetry display (or idle)
    4. Confirm grid shows the final simulation state (not replay state)
    5. Confirm `r` key works to start a new replay

    **Test 5: Disabled interactions**
    1. During replay, click an agent cell
    2. Confirm notification "Interviews unavailable during replay" appears
    3. During replay, press `s` (save)
    4. Confirm notification "Save unavailable during replay" appears

    **Test 6: CyclePickerScreen (if multiple cycles)**
    1. If multiple completed cycles exist, press `r` from COMPLETE state
    2. Confirm CyclePickerScreen overlay appears with cycle list
    3. Use Up/Down to navigate, Enter to select, verify replay starts
    4. Test Esc to cancel -- verify return to COMPLETE state
