    1. `async def compute_influence_edges(self, cycle_id: str, up_to_round: int, total_agents: int) -> dict[str, float]`:
       - Call `session.execute_read(self._read_citation_counts_tx, cycle_id, up_to_round)` to get per-agent citation counts.
       - Compute normalized weights: `weight = citation_count / total_agents` (per D-01).
       - If no citations found, log a warning and return empty dict (per Pitfall 1 in RESEARCH.md).
       - Build edge list: for every agent who was cited, create an edge dict `{"source_id": citing_agent_id, "target_id": cited_agent_id, "weight": weight}`. Actually, per D-01, the weight is on the CITED agent (receiver-centric), not per citing-cited pair. The INFLUENCED_BY edge represents "this agent is influenced by that agent." So for each Decision that CITED an Agent, create an edge from the decision's author to the cited agent with the cited agent's normalized weight.
       - ALTERNATIVE simpler approach per RESEARCH.md recommendation: compute_influence_edges writes Agent-to-Agent edges where each agent that was cited gets INFLUENCED_BY edges FROM every agent that cited them. Weight is the cited agent's normalized score. This requires a second Cypher query to also get the citing agents.
       - SIMPLEST approach: Read all (citing_agent_id, cited_agent_id) pairs from CITED edges, compute per-cited-agent weight = total citations received / total_agents, then write one INFLUENCED_BY edge per unique (citing, cited) pair with the cited agent's global weight.
       - Call `session.execute_write(self._write_influence_edges_tx, edges, cycle_id, up_to_round)` to batch-write.
       - Return the weights dict `{agent_id: weight}` for downstream peer selection (avoids Neo4j re-query per D-06 recommendation).
       - Wrap in try/except Neo4jError -> Neo4jWriteError at domain boundary.

    2. `@staticmethod async def _read_citation_counts_tx(tx, cycle_id, up_to_round) -> list[dict]`:
       Cypher query:
       ```
       MATCH (d:Decision)-[:CITED]->(a:Agent)
       WHERE d.cycle_id = $cycle_id AND d.round <= $up_to_round
       AND d.cycle_id IN [(d)<-[:MADE]-(author:Agent) | author.id][0] <> a.id
       RETURN a.id AS agent_id, count(d) AS citation_count
       ```
       Actually, simpler self-citation filter: use a subquery or two-hop match.
       Better approach:
       ```
       MATCH (author:Agent)-[:MADE]->(d:Decision)-[:CITED]->(cited:Agent)
       WHERE d.cycle_id = $cycle_id AND d.round <= $up_to_round
         AND author.id <> cited.id
       RETURN cited.id AS agent_id, count(d) AS citation_count
       ```
       This filters self-citations (author <> cited) and counts per cited agent cumulatively.

    Full `compute_influence_edges` implementation:
    ```python
    async def compute_influence_edges(
        self, cycle_id: str, up_to_round: int, total_agents: int,
    ) -> dict[str, float]:
        """Compute and write INFLUENCED_BY edges from CITED patterns (D-01, D-02, D-03, D-04).

        Returns weight dict {agent_id: normalized_weight} for downstream peer selection.
        """
        try:
            async with self._driver.session(database=self._database) as session:
                # Read citation counts (cumulative up to round per D-04)
                citation_counts = await session.execute_read(
                    self._read_citation_counts_tx, cycle_id, up_to_round,
                )
                if not citation_counts:
                    self._log.info("no_citations_found", cycle_id=cycle_id, up_to_round=up_to_round)
                    return {}

                # Compute normalized weights (D-01: weight = citations / total_agents)
                weights = {
                    row["agent_id"]: row["citation_count"] / total_agents
                    for row in citation_counts
                }

                # Read citing pairs for edge creation
                pairs = await session.execute_read(
                    self._read_citation_pairs_tx, cycle_id, up_to_round,
                )

                # Build edge list with cited agent's weight
                edges = [
                    {"source_id": p["source_id"], "target_id": p["target_id"], "weight": weights.get(p["target_id"], 0.0)}
                    for p in pairs
                ]

                # Batch-write INFLUENCED_BY edges
                if edges:
                    await session.execute_write(
                        self._write_influence_edges_tx, edges, cycle_id, up_to_round,
                    )

            self._log.info(
                "influence_edges_computed",
                cycle_id=cycle_id,
                up_to_round=up_to_round,
                edge_count=len(edges),
                agents_with_influence=len(weights),
            )
            return weights

        except Neo4jError as exc:
            raise Neo4jWriteError(
                f"Failed to compute influence edges for cycle {cycle_id} round {up_to_round}",
                original_error=exc,
            ) from exc
    ```

    ```python
    def compute_bracket_summaries(
        agent_decisions: list[tuple[str, AgentDecision]] | tuple[tuple[str, AgentDecision], ...],
        personas: list[AgentPersona],
        brackets: list[BracketConfig],
    ) -> tuple[BracketSummary, ...]:
        """Compute per-bracket summaries from a round's decisions (D-07, D-08).

        Promoted from cli.py:_aggregate_brackets() pattern. Excludes PARSE_ERROR agents.
        Returns tuple of BracketSummary for each bracket (preserves bracket order).
        """
        agent_bracket: dict[str, str] = {p.id: p.bracket.value for p in personas}
        display_lookup: dict[str, str] = {b.bracket_type.value: b.display_name for b in brackets}

        # Accumulators per bracket
        counts: dict[str, dict[str, int]] = {}
        conf_sums: dict[str, float] = {}
        sent_sums: dict[str, float] = {}
        totals: dict[str, int] = {}
        for b in brackets:
            bv = b.bracket_type.value
            counts[bv] = {"buy": 0, "sell": 0, "hold": 0}
            conf_sums[bv] = 0.0
            sent_sums[bv] = 0.0
            totals[bv] = 0

        for agent_id, decision in agent_decisions:
            if decision.signal == SignalType.PARSE_ERROR:
                continue
            bv = agent_bracket.get(agent_id)
            if bv is None or bv not in counts:
                continue
            signal_key = decision.signal.value  # "buy", "sell", "hold"
            counts[bv][signal_key] = counts[bv].get(signal_key, 0) + 1
            conf_sums[bv] += decision.confidence
            sent_sums[bv] += decision.sentiment
            totals[bv] += 1

        result: list[BracketSummary] = []
        for b in brackets:
            bv = b.bracket_type.value
            t = totals[bv]
            result.append(BracketSummary(
                bracket=bv,
                display_name=display_lookup.get(bv, bv),
                buy_count=counts[bv]["buy"],
                sell_count=counts[bv]["sell"],
                hold_count=counts[bv]["hold"],
                total=t,
                avg_confidence=conf_sums[bv] / t if t > 0 else 0.0,
                avg_sentiment=sent_sums[bv] / t if t > 0 else 0.0,
            ))
        return tuple(result)
    ```

    3. `select_diverse_peers` pure function (per D-06):
    ```python
    def select_diverse_peers(
        agent_id: str,
        influence_weights: dict[str, float],
        personas: list[AgentPersona],
        limit: int = 5,
        min_brackets: int = 3,
    ) -> list[str]:
        """Select top-N peers with bracket diversity guarantee (D-06).

        Algorithm:
        1. Exclude self from candidates.
        2. Group by bracket, sort each group by dynamic weight descending.
        3. Sort brackets by their top agent's weight.
        4. Phase 1: Pick top-1 from top-min_brackets brackets.
        5. Phase 2: Fill remaining slots by pure weight (any bracket).

        If fewer than min_brackets have candidates, fills from available brackets
        (best effort, always returns up to `limit` peers).
        """
        from collections import defaultdict

        candidates = [p for p in personas if p.id != agent_id]

        bracket_groups: dict[str, list[tuple[str, float]]] = defaultdict(list)
        for p in candidates:
            w = influence_weights.get(p.id, 0.0)
            bracket_groups[p.bracket.value].append((p.id, w))

        for bracket in bracket_groups:
            bracket_groups[bracket].sort(key=lambda x: x[1], reverse=True)

        sorted_brackets = sorted(
            bracket_groups.keys(),
            key=lambda b: bracket_groups[b][0][1] if bracket_groups[b] else 0.0,
            reverse=True,
        )

        selected: list[str] = []

        # Phase 1: top-1 from top-min_brackets brackets
        for bracket in sorted_brackets:
            if len(selected) >= min_brackets:
                break
            if bracket_groups[bracket]:
                agent, _ = bracket_groups[bracket].pop(0)
                selected.append(agent)

        # Phase 2: fill remaining by pure weight
        remaining: list[tuple[str, float]] = []
        for bracket in bracket_groups:
            remaining.extend(bracket_groups[bracket])
        remaining.sort(key=lambda x: x[1], reverse=True)

        for agent, _ in remaining:
            if len(selected) >= limit:
                break
            if agent not in selected:
                selected.append(agent)

        return selected
    ```
